# ID Registry System  
**ระบบจัดการข้อมูลบุคคลสำหรับงานภาคสนาม / ศูนย์พักพิง / งานลงทะเบียนชั่วคราว**
ออกแบบเพื่อช่วยให้เจ้าหน้าที่สามารถบันทึกข้อมูลบุคคลได้อย่างมีประสิทธิภาพเพื่อดำเนินการให้ความช่วยเหลือตามวัตถุประสงค์

  - รวบรวมและประมวลผลข้อมูล
  - ระบบทีมงานพร้อมการกำหนดสิทธิ์
  - รองรับการเพิ่มข้อมูลจากเครื่องอ่านบัตร และแมนนวล
  - เข้ารหัสข้อมูลสำคัญก่อนบันทึกสู่ฐานข้อมูล
  - สามารถเพิ่มฟิลข้อมูลที่ต้องการได้ตามต้องการ
  - ฟังชันค้นหาครอบคลุมเงื่อนไขต่าง ๆ พร้อมปุ่มนำเข้า ส่งออก Excel
  - ตรวจสอบประวัติ log

## 🧭 ระบบประกอบด้วย 2 ส่วนหลัก
  - 🌐 Web Application สำหรับจัดการข้อมูลด้วย UI ที่เป็นมิตรกับผู้ใช้
  - 🪪 Smart Card ID Bridge โปรแกรมอ่านบัตรประชาชน ทำงานผ่าน Local API

## ⚙️ ความต้องการของระบบ (Requirements)
  - PHP 8.1 ขึ้นไปโดยทำการทดสอบบน php 8.3.28
  - PHP extensions `pdo_mysql`, `openssl`, `mbstring` จาก WAMP/XAMPP มาตรฐาน
  - ฐานข้อมูล MySQL / MariaDB
  - เปิด `AllowOverride All` เพื่อให้ `.htaccess` เข้าถึง `.env`
  - Composer 2.x

## 📦 การติดตั้ง
1. เตรียมความพร้อมฝั่งเซิร์ฟเวอร์ด้วยโปรแกรม WAMP, XAMPP, etc พร้อมสร้างฐานข้อมูลและโฟลเดอร์โปรเจค
2. ดาวน์โหลดไฟล์สำหรับติดตั้งเหล่านี้
  - `web` คัดลอกและเปลี่ยนชื่อก่อนนำไปวางแทนที่ `www/project`
  - `installer/id-bridge` คัดลอกยังตำแหน่งที่ต้องการ คลิกขวาที่ไฟล์ setup.bat ด้วยสิทธิ์ Admin เพื่อติดตั้ง
  - สร้างโฟลเดอร์ uploads ภายใต้ `www/project`
3. ติดตั้ง Composer & Library
  - ดาวน์โหลดตัวติดตั้ง `Composer https://getcomposer.org/Composer-Setup.exe`
  - หน้า Installation Options → เลือก **Install for all users**
  - หน้า Settings Check → ช่อง **Command-line PHP** ให้ชี้ไปที่ `C:\wamp64\bin\php\php8.3.28\php.exe` (เปลี่ยนตามเวอร์ชันที่ใช้อยู่)
  - ถ้าเตือน openssl → เปิด `php.ini` แล้วลบ `;` หน้าบรรทัด `extension=openssl` แล้วกดตรวจสอบอีกครั้ง
  - หน้า Proxy Settings → ปล่อยว่างไว้ แล้วกด Next จนจบ
  - ปิด PowerShell ที่เปิดค้างไว้ทั้งหมด แล้วเปิดใหม่
  - ป้อนคำสั่งเช็คสถานะ `composer -V`     # ต้องขึ้นเลขเวอร์ชัน เช่น Composer version 2.7.x
  - ป้อนคำสั่งเช็คสถานะ `php -v`          # ต้องได้ PHP 8.1 ขึ้นไป
  - ติดตั้ง Library ด้วยคำสั่ง

```powershell
$env:PATH = 'C:\wamp64\bin\php\php8.3.28;' + $env:PATH
cd C:\wamp64\www\project
composer install
```

> ถ้า PHP ต่ำกว่า 8.1 Composer จะหยุดพร้อมข้อความ `requires php (^8.1) failed` — ให้เปลี่ยนไปใช้ PHP 8.1+ 

4. เข้าสู่หน้าติดตั้ง `http://localhost/project/install/` แล้วดำเนินการติดตั้งหรืออัพเดตให้แล้วเสร็จ
  - **ติดตั้งครั้งแรก** — สร้างฐานข้อมูล + บัญชีผู้ดูแล + ไฟล์ `.env` (สุ่ม `ENCRYPTION_KEY` ให้อัตโนมัติ)
  - **อัพเดตจากเวอร์ชันเก่า** — สำรองฐานข้อมูลอัตโนมัติแล้วปรับโครงสร้างบนข้อมูลเดิม (สำรอง `.env` ไว้ด้วย)
  - **MySQL ใช้พอร์ต: 3306, MariaDB ใช้พอร์ต: 3307**
5. เมื่อติดตั้งเสร็จ ควรลบโฟลเดอร์ `install/` ออกเพื่อความปลอดภัย

> 🔑 ไฟล์ `.env` เก็บกุญแจเข้ารหัส `ENCRYPTION_KEY` หากหายหรือถูกเขียนทับ = ถอดรหัสข้อมูลเดิมไม่ได้ ควรสำรองแยกไว้เสมอ

## การเพิ่มข้อมูล
1. คลิก "ลงทะเบียนใหม่" เพื่อเข้าสู่หน้าเพิ่มข้อมูลแบบแมนนวล
2. คลิก "อ่านข้อมูลจากบัตรประชาชน" ประมาณ 5-10 วินาที ระบบจะดึงข้อมูลมาแสดง
3. ทำการสอบถามข้อมูลเพิ่มเติมเช่น เบอร์ติดต่อ, อาการเจ็บป่วย
4. ด้านล่างสุดกดปุ่ม "ส่งข้อมูลไปที่แท็บเล็ต" (PDPA สามารถปิดได้กรณีต้องการความรวดเร็วในพื้นที่เสี่ยง)
5. เมื่อผู้เข้าพักกดยินยอม ระบบจะเลือกเช็คบ็อกให้อัตโนมัติ ให้ตรวจทานและบันทึกข้อมูลเป็นลำดับต่อไป
6. ในหน้าประวัติส่วนบุคคล เจ้าหน้าที่สามารถ พิมพ์, แก้ไข, ลบ, และลงทะเบียนออก ผู้พักได้

### การส่งข้อมูลเพื่อให้ผู้เข้าพักยืนยันผ่านหน้าจอมือถือหรือแท็บเบล็ต
  - ทั้งคอมพิวเตอร์และอุปกรณ์ของผู้พักต้องเชื่อมต่อ WiFi เดียวกัน ดังนั้นให้เจ้าหน้าที่เปิด Free Wifi ไว้ 1 ช่องทาง
  - ตั้งค่า Static IP เพื่อป้องกัน Router เปลี่ยนเลข IP เครื่องคอมฯ
  - ตั้งค่า Firewall เปิดพอร์ต: 80 หรือ 8080 เพื่ออนุญาตให้มือถือ "มองเห็น" เว็บเซิร์ฟเวอร์
  - คลิกปุ่ม "สร้าง QR Code" ในหน้า Dashboard แล้วปริ๊นไว้ให้ผู้พักสแกน (ผู้ดูแลแต่ละคนจะมี SEED ID คนละชุด ดังนั้นให้ปริ๊นของตนจึงจะสามารถเชื่อมต่ออุปกรณ์สำเร็จ)
  - กรอกเฉพาะ IP/path เช่น `http://192.168.1.50/project/pages/guest_display.php?d=SEED ID`

### ปัญหาหน้าจอมือถือแจ้งข้อความ "You don't have permission"
  - คลิกซ้ายที่ไอคอน WAMP (มุมล่างขวา) เลือก Apache > httpd-vhosts.conf ทำการแก้ไขและ Restart All Services

```
# Virtual Hosts
#
<VirtualHost _default_:80>
  ServerName localhost
  ServerAlias localhost
  DocumentRoot "${INSTALL_DIR}/www"
  <Directory "${INSTALL_DIR}/www/">
    Options +Indexes +Includes +FollowSymLinks +MultiViews
    AllowOverride All
    # เปลี่ยนจาก Require local เป็นบรรทัดล่างนี้
    Require all granted
  </Directory>
</VirtualHost>
```
