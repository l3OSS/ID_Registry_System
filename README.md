# ID_Registry_System
*ระบบจัดการข้อมูลส่วนบุคคลภายในศูนย์พักพิงชั่วคราว*

ระบบช่วยบันทึกข้อมูลผู้เข้าพักภายในศูนย์ต่าง ๆ ด้วยเครื่องอ่านบัตร และรองรับการเพิ่มข้อมูลแบบแมนนวล เพื่อลดขั้นตอนสำหรับเจ้าหน้าที่ นอกจากนี้ยังสามารถปรับแต่งเป็นระบบบันทึกข้อมูลในโอกาสต่าง ๆ

## ประกอบด้วย
1. ระบบทีมงานและการจำแนกสิทธิ์ตามลำดับขั้น
2. ระบบสรุปผลหน้าแรก Dashboard
3. ระบบแสดงรายชื่อผู้เข้าพักตามเงื่อนไข และส่งออกเป็นไฟล์ Excel
4. หน้าประวัติส่วนบุคคล แสดงข้อมูลการพักและข้อมูลทางการแพทย์ตามที่กำหนด
5. หน้าตั้งค่าการเก็บข้อมูลเพิ่มเติมพร้อมกำหนดให้เป็นตัวเลือกการค้นหา
6. หน้าแสดงประวัติการกระทำต่าง ๆ
7. ระบบเข้ารหัสข้อมูลเลขบัตรประชาชน และเบอร์ติดต่อ
8. เก็บภาพไว้ภายในโฟลเดอร์โปรเจค

## การติดตั้ง
1. ติดตั้งโปรแกรมจำลองเซิร์ฟเวอร์
2. ติดตั้งเครื่องอ่านบัตรประชาชนพร้อมไดรเวอร์
3. ดาวโหลดซอร์สโค้ดทั้งหมด
4. สร้างโฟลเดอร์โปรเจค เช่น www/Reg แล้วคัดลอกไฟล์ทั้งหมดมาเก็บไว้
5. ติดตั้งไลบรารีผ่าน Command Prompt
- เปิด Terminal พิมพ์คำสั่ง cd ตามด้วยที่อยู่โปรเจค เช่น 
`cd C:\wamp64\www\Reg`
- รันคำสั่ง
 `composer install`
6. สร้างฐานข้อมูล
7. เข้าหน้าติดตั้ง 'http://localhost/Reg/install/index.php'
8. หลังติดตั้งเสร็จในหน้า Dashboard มุมล่างขวาจะมีปุ่มดาวโหลดโปรแกรมสำหรับอ่านบัตร
9. คลายซิบไฟล์แล้วรัน setup.bat ด้วยสิทธิ์แอดมินเพื่อติดตั้งตัวโปรแกรม
